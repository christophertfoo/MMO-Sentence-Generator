Welcome to the MMO-Sentence-Generator!

The MMO-Sentence-Generator is a Java application that translates common sentences from massively multiplayer online games (MMOs or MMOGs) from a Neo-Davidsonian semantic representation into an English sentence. Currently, the application can only process a handful of sentence types, but the system can be extended to accommodate even more kinds of sentences.
